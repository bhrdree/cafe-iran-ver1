# Cafe Iran - نسخه ۱.۰

## معرفی پروژه
یک صفحه وب ساده و تک‌صفحه‌ای برای معرفی کافه ایران. این پروژه اولین تجربه‌ی طراحی وب و آشنایی با مفاهیم پایه‌ی HTML است.

## ویژگی‌ها
- ساختار کاملاً HTML
- طراحی مینیمال و ساده
- ریسپانسیو (سازگار با صفحه‌نمایش‌های مختلف)
- بارگذاری سریع و بهینه

## ساختار فایل‌ها

## تکنولوژی‌های استفاده‌شده
- HTML5

## نحوه‌ی اجرا
برای مشاهده‌ی پروژه کافی است فایل `index.html` را در مرورگر خود باز کنید.
